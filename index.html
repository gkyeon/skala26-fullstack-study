🛠 새 강의 추가하는 법

강의가 끝날 때마다 아래 순서로 하면 사이트가 계속 늘어나요. (팀원 누구나 가능)

  1. lectures/ 폴더 안에 새 폴더 생성 (예: lectures/02-backend-basics/)
  2. template/lecture-template.html 을 복사해서 새 폴더의 index.html 로 저장
  3. 템플릿 안 주석(<!-- 여기에 -->)을 따라 강의 내용 · 실습 · 퀴즈를 채워넣기 (Claude에게 PDF/자료 주고 "이 템플릿 형식으로 채워줘"라고 요청해도 됨)
  4. 이 index.html(홈)의 강의 목록 <div class="grid"> 안에 새 카드 하나 추가하고, "준비 중" 카드는 지우거나 다음 자리로 미루기
  5. git add . && git commit -m "add: 새 강의 추가" && git push — GitHub Pages가 몇 분 안에 자동 반영
